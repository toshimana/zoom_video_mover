# ISO 25010 品質特性レビュー - Zoom Video Mover

## 文書概要

**プロジェクト名**: Zoom Video Mover
**レビュー日**: 2026-02-19
**レビュー対象バージョン**: v0.1.0
**レビュー実施者**: 品質保証チーム
**ISO規格**: ISO/IEC 25010:2011 (SQuaRE)

### 目的

本文書は、ISO 25010 品質モデルに基づき、Zoom Video Mover プロジェクトの品質特性を体系的にレビューする。製品品質モデル（8特性31副特性）および利用時の品質モデル（5特性10副特性）の全41副特性について、プロジェクト固有の検討項目を整理し、対応状況を可視化する。

### 対応状況の判定基準

| 記号 | 判定 | 基準 |
|------|------|------|
| ◎ | 対応済み (Covered) | 明示的記載あり＋定量基準あり＋設計/実装トレースあり |
| ○ | 部分対応 (Partial) | 関連記述あり、ISO観点で不足あり or 定性的のみ |
| △ | 未対応 (Gap) | 該当記述なし、検討必要 |
| - | 適用外 (N/A) | プロジェクト特性上該当なし（理由明記） |
